# LSS_BOT_2025
how to install & run the bot
1. install chrome
2. install python
3. install git
4. install LSS_BOT_2025
5. config (setup Username, Password, Webhook_URL)
6. open cmd inside of LSS_BOT_2025 (cd folder_path or click on the address bar and type in cmd)
7. type in "python -m venv venv"
8. type in ".\venv\Scripts\activate"
9. type in pip install selenium
10. type in pip install requests
11. type in python start_bot.py
